# Recursos Interactivos (UNAM SUAyED Mathematics)

[![Live Demo](https://img.shields.io/badge/View-Live_Demo-informational)](https://luisdmares.github.io/Recursos-interactivos/)

## Overview
A collection of interactive digital tools developed for the Distance Learning Mathematics program (SUAyED) at the National Autonomous University of Mexico (UNAM). This project aims to bridge the gap in remote technical education by providing dynamic visual aids for complex mathematical analysis and calculus concepts.

## Tech Stack
* **Web Technologies:** HTML5, CSS3, Vanilla JavaScript
* **Mathematical Visualization:** Canvas API / SVG manipulation (if applicable)

## Key Technical Features
* **Dynamic Function Manipulation:** Interactive modules allowing students to visualize mathematical transformations, such as real-time function translations across coordinate planes.
* **Responsive Pedagogy:** Tools designed to scale across devices, ensuring accessibility for distance-learning students.
* **Parametric Controls:** UI sliders and inputs directly bound to underlying mathematical equations to demonstrate immediate graphical output.

## Impact
Developed as part of academic community service, these resources directly support students in building intuition for formal mathematical reasoning, serving as a core supplementary tool for the university's remote curriculum.
